List of File Formats

There are many different file formats that are recognized by various programs. This is a (currently partial) list of them, organized alphabetically by file name extension. Note that I have made an effort to encompass only the most common files.

.7z7-Zip compression
.binAn executable (generalized)
.bmpBitmap (Windows)
.cC source
.cssCSS stylesheet
.docxMicrosoft Word document
.debDebian-base Linux package file
.dmgApple application
.exeWindows program
.gzG-Zipped file
.htmlHTML file
.isoISO9660 optical disk image
.jsJavaScript file
.jpgJPEG image
.mdMarkDown document
.pdfPortable Document Format
.pngPortable Network Graphics image file
.pyPython file
.romROM file
.tarTarball (tar program)
.txtText file
.xlsxMicrosoft Excel spreadsheet
.xmleXtensible Markup Language
.xzLZMAL
.zipCommon compression format
